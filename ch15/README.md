## 15장 Context API

Context API : 전역 데이터를 관리하는 기능
- 리덕스, 리액트 라우터, styled-components 등의 라이브러리는 Context API를 기반으로 구현

리액트는 주로 최상의 컴포넌트인 App의 state에 넣어서 전역적으로 필요한 상태를 관리한다.<br>
하지만 많은 컴포넌트를 거쳐야 하며, 다룰 데이터가 훨씬 많아질 수 있어 유지 보수성이 낮아질수 있다.<br>
따라서 리덕스, MobX 같은 상태 관리 라이브러리를 사용해 전역 상태 관리 작업을 편하게 처리하며,<br>
Context API 역시 많이 개선되어 별도의 라이브러리 없이 전역 상태 관리가 가능하다.
